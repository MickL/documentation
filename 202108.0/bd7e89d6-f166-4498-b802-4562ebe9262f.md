The *Category Management* feature allows Back Office users to create parent and child categories, which can be shared between stores. A category is a group of products that share a common attribute and, therefore, can be logically united. Thus, Back Office users can structure products into a logical system where each product belongs to a category or a set of interrelated categories and, this way, improve navigation and product search for customers on the Storefront.
